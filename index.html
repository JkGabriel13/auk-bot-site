Bem-vindo ao Meu Painel do Auk Bot

Este bot foi criado para ajudar você a gerenciar seu servidor do Discord de forma divertida e eficiente. Com uma variedade de comandos e funcionalidades, ele pode tornar sua experiência no Discord ainda melhor!

Funcionalidades

  • 🎮 Jogos interativos
  • 🔧 Comandos de moderação
  • 🤖 Respostas automáticas personalizáveis
  • 🌐 Integração com APIs externas
  • 📊 Comandos de estatísticas e relatórios

Como adicionar o bot ao seu servidor

Clique no botão abaixo para convidar o bot para o seu servidor:

Adicionar Bot

Instruções de Uso

Após adicionar o bot ao seu servidor, você pode usar os seguintes comandos:

  • a-help - Mostra todos os comandos disponíveis.
  • a-work - Começe A Trabalhar Para Juntar Moedas
  • a-daliy - Para Pegar Seu Daily Diario.
